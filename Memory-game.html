    • Memory Game

    Breadboards, Circuits, Arduino

    The course (Intro to Robotics) objective of this project was to create an electro-mechanical game. As a personal objective, I wanted to create a game which was heavily dependent on coding, as it is my area of greatest desired improvement in engineering.

    This memory game flashes an LED pattern. The user can then repeat the pattern back by pressing the corresponding buttons. If the player is correct, all the LEDs will flash and another pattern will display. If the player is incorrect, the red LED will flash and repeat the pattern.
