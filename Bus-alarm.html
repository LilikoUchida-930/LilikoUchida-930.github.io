    • Bus Arrival Alarm Clock 2019

    3D printing, Laser cutting, MBTA API

    For this project, my team and I created an alarm clock which rings the trumpet horn at the arrival of the local bus. We used the MBTA API to grab information about the current bus location and created a program in LabVIEW which vibrates the bell when the bus arrives.
