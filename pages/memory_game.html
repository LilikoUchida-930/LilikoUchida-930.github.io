Memory Game



For project 3 of my Robotics and Electromechanical Systems course, I designed and built a memory game. Once the game begins, a random pattern lights on the LEDs. After 5 seconds, the player is allowed to repeat the pattern to their best recollection by pressing the corresponding colored buttons. If the player is correct, all LEDs will flash and a motor with streamers will spin to celebrate. If the player is wrong, the red LED will flash. Find the code here.
