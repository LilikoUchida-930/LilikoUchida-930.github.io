mathematical model

To begin, I selected mechanism #145 from "507 Mechanical Movements". I then decided to model the velocity of slider B as it relates to the change in θ for the circular fly wheel. The image to the right shows my diagram of the mechanism along with assumptions being made, trigonometric relations within the mechanism, and calculations which ultimately resulted in an expression for the velocity of slider B in terms of any given function for θ.

plotting with python

Next, I plotted the velocity of slider B against time with a given function for theta (theta = 0.01t^2 for this example, but the user may insert any function for theta in the code) using Matplotlib. The resultant plot is shown below and the code can be found in my Github here.







CAD model, motion study, high-fidelity prototyping

The final step to this project was to create a CAD model of my mechanism (images pending), perform a simple motion study and produce a plot to compare to the plot built from the math model (images pending), and finally to create a high-fidelity prototype. I made my prototype with acrylic.
